# 字幕系統更新說明（2026-04-21）

嗨剪輯團隊 👋 這次更新主要解決兩個大家回報的問題：

1. **預先設定的關鍵字（詞庫）不一定會出現** ← 這次可以自己調嚴格度
2. **字幕時間跟影片對不太上** ← 這次自動變得更精準（不用你操作）

下面用最白話的方式說明怎麼用。

---

## 🎬 新功能一：字幕時間自動對準

### 你會看到什麼
字幕時間會更貼合語句的實際開始與結束，對字幕不再要手動微調那麼多。

### 要做什麼
**什麼都不用做**，系統後台自動換了更精準的時間辨識方式，下次辨識就會套用。

---

## 🎛️ 新功能二：三檔「詞庫嚴格度」可以選

上傳影片的頁面多了一個選項：**4. 詞庫比對嚴格度**

### 三個選項

| 選項 | 什麼時候用 | 會發生什麼 | 費用 |
|---|---|---|---|
| 🟢 **寬鬆** | 影片主題很雜、不想被硬改 | 只有在辨識時「提示」Whisper 可能會出現哪些詞，不事後替換 | 免費 |
| 🟡 **正常**（預設） | 多數情況用這個 | 辨識完**只替換你事先填好的「常見錯字」**。例如你把「梅卡巴」填進「梅爾卡巴」的錯字清單，辨識出「梅卡巴」就會被自動換成「梅爾卡巴」 | 免費 |
| 🔴 **嚴格** | 主題很明確、一定會一直講專有名詞（例：占星教學、脈輪課程） | 把辨識結果**整篇丟給 GPT-4 重新校對**，AI 會看你完整的詞庫，把聽起來很像但寫錯的字全部改對 | 每小時影片約 + 台幣 $0.5 |

### 怎麼挑？
- 不確定就選 **🟡 正常**
- 怕 AI 亂改、想保留原本辨識結果就選 **🟢 寬鬆**
- 精準度要求很高的成品就選 **🔴 嚴格**

## 📖 新功能三：詞庫可以加「常見錯字」

### 為什麼要填
「正常」和「嚴格」兩個模式都需要你告訴系統「這個詞常被聽錯成什麼樣子」，系統才知道要把哪些錯字改掉。

### 怎麼填

進 **詞庫管理** → 選一個分類 → 你會看到每個詞變成**一行一格**的卡片。

每個詞底下有一個輸入框，寫「這個詞常被聽錯成什麼」。

例：

```
正確寫法：梅爾卡巴
常見錯字：梅卡巴、梅卡巴巴
```

```
正確寫法：阿卡夏記錄
常見錯字：阿卡西記錄、阿卡莎記錄
```

### 小技巧
- 按 **Enter** 或 **逗號** 會自動建立一個錯字
- 可以一次貼很多個，用「,」、「、」或換行分開
- **左邊 backspace** 可以刪掉最後一個
- 每個錯字旁邊有 × 可以單獨刪掉
- 改完記得按「儲存」

### 建議做法
1. 先挑幾支以前做過的影片
2. 看字幕裡常被辨識錯的詞
3. 把錯法填進去
4. 下次辨識時就會自動改對

## 💡 常見問題

### Q：我如果只選「寬鬆」，詞庫還有意義嗎？
有！Whisper 辨識時還是會看到詞庫提示，只是辨識完不會再做第二層替換。

### Q：「嚴格」模式會不會把對的東西亂改？
系統有下指令只改「幾乎同音」的詞，其他一字不動。但若你詞庫填得太雜，還是有可能誤判，建議：
- 影片主題明確時才用嚴格模式
- 詞庫裡不要混入太多「可能會跟日常語混淆的」詞

### Q：「嚴格」模式的錢算誰的？
走你的 OpenAI 帳號，和 Whisper 辨識同一個 API key。

### Q：如果一個詞我還沒填「常見錯字」會怎樣？
「正常」模式下這個詞就不會被替換。「嚴格」模式下 AI 可以從整段語境自己判斷要不要改，但建議還是手動填幾個已知錯字比較保險。

### Q：我看到「原始逐字稿」是什麼？
系統現在會同時存「原始辨識結果」和「修正後結果」。之後會做個「對照頁」讓你看兩邊差異（下一次更新加）。
